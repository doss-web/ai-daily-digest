💡 Today's Take

The most significant signal today is the **paradigm shift from "using models" to "training skills" in the agent ecosystem**. Dozens of agent skills repositories emerged on GitHub in a single day (agent-skills, claude-skills, taste-skill), and Microsoft open-sourced SkillOpt making skills trainable, verifiable, and deployable.
