📄 重要论文

GRPO、Dr. GRPO 和 DAPO 本质是同一个操作
论文证明三种最流行的推理训练方法实际上都在调节同一个数值——组标准差，即同一提示下各采样答案的分歧程度。当答案均匀分裂时标准差最大，完全一致时为零。
★★★★★ 统一理解推理训练方法，简化模型调优策略
HuggingFace Papers
更多采样反而有害：测试时扩展的模态天花板和相关性天花板
研究发现测试时扩展（多次采样）虽然提高了覆盖率，但选择正确答案的能力受限于模态天花板和相关性天花板，过度采样反而会降低最终答案质量。
★★★★★ 揭示推理系统采样的根本限制，指导实际部署策略
HuggingFace Papers
PACE：用低成本代理指标预测 Agent 能力
论文提出 PACE，通过少量非 Agent 性基准（推理、代码生成）的分数，高效预测 SWE-Bench、GAIA 等昂贵 Agent 基准的表现，单次评估成本降低 100 倍以上。
★★★★★ 大幅降低 Agent 评估成本，加速模型迭代
HuggingFace Papers
WARP：从权重空间恢复训练数据组成
论文提出 WARP 方法，仅通过模型权重就能推断出训练数据的领域混合比例，无需访问训练数据或训练过程。
★★★★★ 提升模型透明度，帮助研究者理解训练数据分布
HuggingFace Papers
AutoMem：将记忆管理作为可训练技能
论文将文件系统操作提升为与任务动作并列的一等记忆动作，让模型自主决定如何管理记忆，在长上下文任务上取得显著提升。
★★★★★ Agent 记忆管理从人工设计走向自主学习

📄 Papers

GRPO, Dr. GRPO, and DAPO Are Three Operations on One Number
Paper proves that three most popular reasoning training methods all regulate the same value — the group standard deviation measuring answer disagreement across samples. Largest when answers split evenly, zero when unanimous.
When More Sampling Hurts: Modal Ceiling and Correlation Ceiling of Test-Time Scaling
Research reveals that while test-time scaling (multiple sampling) improves coverage, the ability to select correct answers is capped by modal and correlation ceilings, with oversampling degrading final answer quality.
PACE: Predicting Agent Capabilities with Low-Cost Proxy Metrics
Paper proposes PACE, using scores from a few non-agentic benchmarks (reasoning, code generation) to efficiently predict performance on expensive agent benchmarks like SWE-Bench and GAIA, reducing evaluation cost by 100x+.
WARP: Recovering Training Data Composition from Weight Space
Paper proposes WARP method that infers domain mixture ratios of training data from model weights alone, without accessing training data or process.
AutoMem: Learning Memory Management as a Trainable Skill
Paper promotes file-system operations to first-class memory actions alongside task actions, letting the model decide how to manage its own memory, achieving significant improvements on long-context tasks.
SkillCoach: Self-Evolving Rubrics for Evaluating Agent Skill Use
Paper proposes SkillCoach, a self-evolving evaluation framework that automatically generates and iteratively improves rubrics to precisely assess agent performance in complex skill repositories.
UK AI Security Institute Finds Standard Benchmarks Systematically Underestimate Agent Capabilities
AISI research shows standard AI evaluations systematically underestimate agent capabilities by capping compute budgets. On software engineering tasks, success rates jumped ~25% when token budget increased 10x, with frontier model actual progress 60% steeper than previously measured.
