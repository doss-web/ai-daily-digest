今天AI圈相当热闹，几个大新闻绝对值得你花几分钟通勤时间来听听。咱们从最重磅的模型发布开始，再到一个差点颠覆统计学的数学发现，还有一大波关于Agent安全、开源工具和开发者灵感的消息。准备好了吗？咱们这就开聊。
先看第一个大新闻。由前OpenAI和Meta研究员创办的Thinking Machines Lab，终于发布了他们的第一个公开模型，名字叫Inkling。这个模型有九千七百五十亿参数，支持文本、图像、视频和音频，是现在最大的开放权重多模态模型之一。他们花了三万五千张H100来训练，开源许可用的是Apache 2.0，听上去很不错。但注意一点，它没包含训练代码、数据，还有强化学习用的协议栈。也就是说，这个模型是开放的，但你想直接用起来做产品部署，还得自己再搭不少东西。不过它在多模态任务上已经接近GPT-5.5的水平，值得关注。
另外一个让人忍不住倒吸一口气的新闻来自OpenAI。他们的最新推理模型，GPT-5.6 Sol，在九十分钟内就推翻了一个三十年的统计学猜想，叫Benjamini-Hochberg方法的核心猜想。宾夕法尼亚大学的一位教授用这个模型给出了一个全新的证明，人类研究者好几年都没解出来，它却用一个组合已知方法的思路，给了一个全新的解答。这事儿太有意义了——模型不仅会搜索知识，开始会“发现新知识”了。
好，搞科研归搞科研，咱开发者还得关注现实中的坑。今天有几个关于Agent安全和成本控制的消息。首先，OpenAI的Codex从六月开始，对主Agent传递给子Agent的指令做了加密传输。也就是说，开发者以后在日志里看不到内部任务怎么委派的。对于调试、安全审计和信任建立，这绝对是个痛点。其次，1Password推出了AI成本管理功能，帮助企业实时监控Anthropic、Cursor、OpenAI这些服务的token支出，防止预算突然失控。看来大公司也终于意识到，Agent用多了，钱花得像流水一样。
开源这块也很精彩。Google正式发布了LiteRT.js，让你能在浏览器里直接用WebGPU运行TensorFlow Lite模型，性能比之前快三到六十倍。同时德国AI联盟发布了Soofi S，一个三十一点六B参数但只激活三点二B参数的混合Mamba-Transformer架构模型，在德语和英语的测试上超过了同尺寸的LLaMA和Mistral，而且是完全开源。还有PrismML把Qwen3.6压到了四个GB，直接可以在手机和笔记本上跑二十七B的模型，苹果据说已经在测试这种极端压缩技术了。
今天还有几个非常实用的开源工具，独立开发者可以多看两眼。一个叫Destructive Command Guard，是个轻量级CLI工具，专门挡Agent执行那些危险的命令，比如rm -rf / 或者 git push --force到错误仓库。你如果天天用Claude Code或者Codex搞代码，这个基本是安全必需品。另一个叫OfficeCLI，专门给Agent读写Word、Excel和PowerPoint文件的，不需要装Office，一个二进制就搞定。另外，VoltAgent社区维护了超过一千个Agent技能集，名字叫awesome-agent-skills，直接导入就能用，省了你自己从头写提示词。
最后，咱们也不能忘了法律风险。26名前Meta员工在加州提起了集体诉讼，指控Meta在二零二二到二零二三年裁员八千人时，用内部AI工具分析绩效数据，系统性地针对正在休病假、产假或残疾假的员工进行裁员。这是首个大型AI驱动裁员的歧视诉讼，如果成立，对企业使用AI做人力资源决策会有很大震慑。
好啦，咱们用今天的深度观察来收个尾。今天这些事件拼在一起，三个趋势很明显：开放模型正从参数竞赛转向实用部署，模型大小不再是唯一指标，本地运行能力成了新焦点；Agent的安全和透明性问题集中爆发，很多人在提醒，Agent的采用速度已经超过了我们对风险的控制能力；最后，科学发现正在成为大模型能力的试金石，GPT-5.6 Sol推翻三十年统计猜想的例子，暗示AI正在从工具变成合作研究者。三个月后回看，今天最重要的可能就是这个数学发现。
