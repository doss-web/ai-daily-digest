**📄 Function-Aware Fill-in-the-Middle as Mid-Training for Coding Agent Foundation Models**
- **What happened**: The authors observe that a coding agent's action-observation-continuation loop is structurally isomorphic to a function call site. The resulting model achieves 45.6% on SWE-bench Verified, surpassing prior methods that fine-tuned on evolution trajectories (38.1%).

**📄 Weak-to-Strong Generalization via Direct On-Policy Distillation**
- **What happened**: Running RL with verifiable rewards (RLVR) on each new strong model is costly. This paper proposes a weak-to-strong alternative: run RL on a smaller model (cheaper rollouts), then distill the learned knowledge to a stronger target model. On-policy distillation significantly outperforms off-policy. The weak-to-strong generalization works effectively and efficiently.
- **Why it matters**: As models scale, post-training becomes a bottleneck. This method allows developers to explore policy spaces with small models and transfer to large ones, drastically reducing post-training costs. This is a practical, deployable cost optimization for training.
