## 📄 Papers

**📄 LongStraw: Million-Token RL Post-Training Under a Fixed GPU Budget**
- **What happened**: LongStraw is an architecture-aware execution stack that enables RL post-training for LLMs with over 2 million token context on a fixed GPU budget (e.g., 8 A100s). It bridges the gap between inference-side million-token contexts and current 256K token RL training, specifically targeting agent tasks that accumulate long trajectories, tool outputs, and documents.
- **Why it matters**: For teams training agent models, this is critical—current RL post-training practices are stuck at ~256K context, while inference has already entered the million-token regime. LongStraw provides concrete engineering to close that gap, enabling models to learn better reasoning over long contexts.

**📄 SEED: Self-Evolving On-Policy Distillation for Agentic Reinforcement Learning**
- **What happened**: SEED addresses the supervision gap in agent RL caused by sparse trajectory-level rewards. It lets a teacher model self-improve through environment feedback, then distill token-level knowledge to a policy model, without requiring stronger models or additional labeling.
- **Why it matters**: Credit assignment in long-horizon tasks is the core bottleneck in agent training. SEED provides a practical self-evolving mechanism to improve open-source agent models on complex tool-use and multi-turn interactions, especially when combined with million-token training like LongStraw.
- Importance: ★★★★★
- Source: [HuggingFace Papers](https://huggingface.co/papers/2607.14777)

**📄 Smarter and Cheaper at Once: Byte-Exact KV-Cache Grafting**
- **What happened**: This paper shows how to make a frozen small model both more capable and cheaper: store verified knowledge once as a byte-exact KV state artifact, then "graft" it into a new inference context. The restored logits are byte-for-byte identical to fresh computation (SHA-256 match, zero KL divergence), while inference cost drops by over 90%.
- **Why it matters**: Developers can precompute an expensive model's reasoning for specific knowledge and reuse it on a lightweight model at negligible cost. For agent systems that frequently query the same knowledge (e.g., customer support, legal Q&A), this can dramatically cut latency and cost. Grafting guarantees identical outputs, making it suitable for deterministic replay scenarios.
- Importance: ★★★★★
- Source: [HuggingFace Papers](https://huggingface.co/papers/2607.14431)

**📄 RoboTTT: Scaling Robot Policy Context to 8K Timesteps**
- **What happened**: RoboTTT (Test-Time-Training Robot Policies) scales visuomotor context to 8,000 timesteps—three orders of magnitude beyond state-of-the-art—without increasing inference latency. This enables one-shot imitation learning from human video demos, on-the-fly policy improvement, and robustness to perturbations.
- **Why it matters**: This is a major breakthrough in robotics: the massive context expansion means robots no longer need per-task training but can learn from long histories like LLMs. For embodied AI teams, this makes "deploy once, adapt via long context" possible and could reshape robot software stack design.
- Importance: ★★★★★
- Source: [HuggingFace Papers](https://huggingface.co/papers/2607.15275)

**📄 Token Time Continuous Diffusion for Language Modeling**
- **What happened**: TTCD (Token Time Continuous Diffusion) is a new diffusion language model that operates in continuous space, deterministically mapping Gaussian noise to token canvases without additional sampling. Its key innovation is per-token independent times—some tokens transition from noise to token faster than others, avoiding inaccuracies from parallel multi-token sampling.
- **Why it matters**: TTCD offers a non-autoregressive paradigm with potential speed and quality advantages. For teams researching new architectures, this is a promising direction. It may alleviate the difficulties autoregressive models face when generating content with long-range structure.
